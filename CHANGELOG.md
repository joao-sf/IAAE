# Changelog

## 1.3.0 — Portfólio público

- consolidação de 15 CATMATs em quatro famílias;
- integração hierárquica dos medidores de energia;
- correção da propagação de Estado/UF até o Power BI;
- modelo final com 2.322 compras, 719 UASGs e 27 UFs;
- previsão com 540 linhas e três cenários;
- documentação e estrutura preparadas para publicação pública;
- remoção de arquivos locais, caches, backups e dados gerados do pacote Git;
- dashboard final mantido como artefato de portfólio.

## 1.2.0 — Python 3.14 e identidade IAAE

- projeto renomeado para IAAE;
- suporte declarado ao Python 3.14;
- ambiente, testes e integração contínua consolidados.
